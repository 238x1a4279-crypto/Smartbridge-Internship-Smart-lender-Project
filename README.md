🏦 Smart Lender – Applicant Credibility Prediction for Loan Approval

📌 Project Overview

Smart Lender is a machine learning web application that predicts whether a loan applicant is likely to be approved based on the information provided by the user. The application uses a Decision Tree Classifier to analyze applicant details and provide instant predictions through a simple and user-friendly web interface.

This project was developed as part of the SmartBridge Internship Program.

---

🎯 Objective

The objective of this project is to assist financial institutions in making faster and more consistent loan approval decisions by leveraging machine learning techniques to evaluate loan applications.

---

🚀 Features

- Predicts loan approval eligibility based on applicant details.
- User-friendly and responsive web interface.
- Machine Learning-powered Decision Tree Classifier.
- Fast and accurate prediction results.
- Built using Flask for seamless web deployment.

---

🛠️ Tech Stack

- Programming Language: Python
- Frontend: HTML, CSS
- Backend: Flask
- Machine Learning: Scikit-learn
- Data Processing: Pandas, NumPy
- Model: Decision Tree Classifier

⚙️ Installation

1. Clone the repository

git clone https://github.com/238x1a4279-crypto/Smartbridge-Internship-Smart-lender-Project.git

2. Navigate to the project directory

cd Smartbridge-Internship-Smart-lender-Project

3. Install the required dependencies

pip install -r requirements.txt

4. Run the Flask application

python app.py

5. Open the application

Visit the following URL in your web browser:

http://127.0.0.1:5000
